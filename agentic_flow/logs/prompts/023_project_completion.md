## Deployment Instructions

### Backend Deployment
```bash
cd backend
pip install -r requirements.txt
python app.py
```
Backend runs on `http://localhost:5000`

### Frontend Deployment
```bash
cd frontend
npm install
npm start
```
Frontend runs on `http://localhost:3000`
